> 구매담당자는 공급업체를 찾고, 공급업체는 구매담당자를 찾습니다.
> **서로 찾고 있지만 아직 만나지 못한 비즈니스 파트너를 동의 기반으로 연결합니다.**

## 💡 핵심 컨셉
기존 서비스와의 차이:
- **리멤버** = 이미 만난 사람 관리 (명함첩)
- **LinkedIn** = 나를 보여주는 SNS
- **BizMatch** = 서로 찾는 사람을 매칭 (양방향)

## ✨ 주요 기능
- 📡 **신호 시스템** — "나는 ___를 찾고 있다" 신호를 켜놓음
- ⚡ **자동 매칭** — 찾는 것 ↔ 찾아지고 싶은 것이 일치하면 매칭
- 🎯 **매칭률 스코어** — AI가 매칭 정확도를 수치화
- 🤝 **양방향 동의** — 양쪽 모두 수락해야 연락처 상호 공개
- 📖 **파트너 관리** — 연결된 비즈니스 파트너 관리
